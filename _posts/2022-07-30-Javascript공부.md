### 변수

* 변수규칙

  * 숫자로생각하면 안됨
  * 특수문자는 _$만 됨
  * 예약어는 안됨
  * let는 변수를 중복해서 선언하는것을 허용하지 않음 (var와의 차이)

  * 상수는 const : 선언하면서 초기화

* 자바스크립트의 자료형

  * primitive type : 한번에 하나의 값. 하나의 고정된 저장공간

  * non-primitive type : 한번에 여러값. 여러개의 고정되지 않은 동적 공간

    ```
     let array = [1,2,3,4];
    ```
